Pollen Markup

  1. start pollen server, local client only! raco pollen start --local

  2. Note, you can enter a different port number:

    3. raco pollen start . 8099
    Note the dot instead of the root directory!

  3. Go to browser an open dashboard http://localhost:<port>/index.ptree
    Click on the file name hello.txt on the list to preview the rendered text file.

    Click the link "in" to preview the source code

  4. Lozenge character is used for pollen commands to be executed, embedded within the text.
    Use editor vim in the terminal selecting the Lozenge-character from Edit menu emoj & symbols
